# Leave Management System (MERN Stack)

This is a simple **Leave Management System** built using the **MERN stack**  
(MongoDB, Express.js, React.js, Node.js).

Users can apply for leave, view their leave history, and see their leave balance.  
Admins can view all pending leave requests and approve or reject them.

## Features

### User
- Login
- View dashboard with leave balance
- Apply for leave
- View leave history
- Logout

### Admin
- Login as admin
- View all pending leave requests
- Approve or reject leave requests
- Leave balance updates automatically after approval

## Tech Stack

- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT (JSON Web Token)

NOTE:- in this project the user and admin are mocked, there is no feature of register a new user. In backend/seed.js the mock users are created.
